\section{What you need?}
\begin{itemize}
  \item OS: GNU/Linux or macOS. Windows is not supported
  \item CUDA SDK 7+.
  \item Compiler: nvcc (bundled with CUDA SDK) and gcc 4.8+ (for C++11 standard support).
  \item libjpeg to encode/decode JPEG files.
\end{itemize}
